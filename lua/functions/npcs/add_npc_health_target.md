
# (Official) LUA Scripting Documentation

## add_npc_health_target
#
### Add or subtract to/from the target NPCs health.
#
Spec:
```lua
add_npc_health_target(
	points,
	millisecs,
	duration)
```
#
## Parameters:
- `points:` How many hitpoints to add. Use a negative number to subtract from (damage) the NPCs health
- `millisecs:` 
- `duration:`
