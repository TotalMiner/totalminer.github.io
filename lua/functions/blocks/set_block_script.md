
# (Official) LUA Scripting Documentation

## set_block_script

Assign a script to a block at a voxel position.

___

Spec:

```lua
set_block_script(
	x,
	y,
	z,
	name,
	type)
```

## Parameters

- `x`: The x component of the position
- `y`: The y component of the position
- `z`: The z component of the position
- `name`: 
- `type`:
