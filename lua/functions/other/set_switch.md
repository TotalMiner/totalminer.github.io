## set_switch

Set a switches on/off position.

___

Spec:

```lua
set_switch(
	long x,
	long y,
	long z,
	bool power)
```

## Parameters

- `x`: The x component of the map point of the switch.
- `y`: The y component of the map point of the switch.
- `z`: The z component of the map point of the switch.
