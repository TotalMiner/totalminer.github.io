
# (Official) LUA Scripting Documentation

## menu

Display a menu (list of items) and wait for user input (item selection).

___

Spec:

```lua
menu(
	string heading,
	object[] args)
```

## Parameters

- `heading`: Header Text
- `args`: A list of items

## Returns

- `long`: The number of the selected menu item. Zero = no selection (escape).
