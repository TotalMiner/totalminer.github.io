
# (Official) LUA Scripting Documentation

## add_skill_xp

Increase or decrease skill xp of the context actor.

___

Spec:

```lua
add_skill_xp(
	skill,
	xp)
```

## Parameters

- `skill`: The skill to change
- `xp`: The amount of xp to add (positive number) or subtract (negative number)
