
# (Official) LUA Scripting Documentation

## set_clan

Assign the context players clan.

___

Spec:

```lua
set_clan(
	string name,
	int banner_id)
```

## Parameters

- `name`: The name of the clan
- `banner_id`: The ID of the clan banner icon (See Clan Banner Screen on Character Tab)
