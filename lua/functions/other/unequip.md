
# (Official) LUA Scripting Documentation

## unequip

Unequip an item.

___

Spec:

```lua
unequip(
	long item_id,
	string slot)
```

## Parameters

- `item_id`: The item to unequip
- `slot`: Slot to unequip from. Valid values `left`, `right`, `body`
