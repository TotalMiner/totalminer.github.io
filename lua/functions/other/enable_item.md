
# (Official) LUA Scripting Documentation

## enable_item

Enable or disable an item.

___

Spec:

```lua
enable_item(
	long item_id,
	bool is_enabled)
```

## Parameters

- `item_id`: The item to enable/disable
- `is_enabled`: true = enable, false = disable

___

## Example

```lua
enable_item(item.minigun)
```
