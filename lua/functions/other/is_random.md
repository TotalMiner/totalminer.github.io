
# (Official) LUA Scripting Documentation

## is_random

A dice query (same as is_chance).

___

Spec:

```lua
is_random(
	long chance,
	long chances)
```

## Parameters

- `chance`: Numbers for a win
- `chances`: Number of sides on the dice

## Returns

- `bool`: True if the dice throw was won

___

## Examples

```lua
is_chance(1,6)
```

___

```lua
is_chance(2,20)
```
