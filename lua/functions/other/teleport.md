
# (Official) LUA Scripting Documentation

## teleport

Teleport the player.

___

Spec:

```lua
teleport(
	x,
	y,
	z)
```

## Parameters

- `x:` The x component of the teleport destination.
- `y:` The y component of the teleport destination.
- `z:` The z component of the teleport destination.
