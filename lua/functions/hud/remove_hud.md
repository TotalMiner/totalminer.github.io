
# (Official) LUA Scripting Documentation

## remove_hud

Remove a hud element.

___

Spec:

```lua
remove_hud(
	string name,
	bool is_player)
```

## Parameters

- `name`: The name of the element to remove. Omit or pass and empty string to remove all elements
- `is_player`: True or False. Omit for default of False
