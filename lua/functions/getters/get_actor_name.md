
# (Official) LUA Scripting Documentation

## get_actor_name

Get the name of the context actor.

___

Spec:

```lua
get_actor_name()
```

## Returns

- `string`: The actor name. The NPC name for NPCs or the gamer name for players.

___

## Example

```lua
notify(get_actor_name().." has completed a level!")
```
