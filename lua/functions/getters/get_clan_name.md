
# (Official) LUA Scripting Documentation

## get_clan_name

Get the clan name of the context actor.

___

Spec:

```lua
get_clan_name()
```

## Returns

- `string`: The clan name. Empty if the actor is not a member of a clan

___

## Example

```lua
notify("The player is a member of clan: ", get_clan_name())
```
