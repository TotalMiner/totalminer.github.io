
# (Official) LUA Scripting Documentation

## get_item

Get an equipped or event raising item.

___

Spec:

```lua
get_item(
	string src)
```

## Parameters

- `src`: The item source. Omit to get the item that caused the script to be executed (via ItemSwing event)

## Returns

- `long`: The item id
