
# (Official) LUA Scripting Documentation

## get_npc_count_in_zone

Get the total number of NPCs currently positioned inside a zone.

___

Spec:

```lua
get_npc_count_in_zone(
	string zone_name,
	string npc_type)
```

## Parameters

- `zone_name`: 
- `npc_type`: The type of NPC to count. Omit to count all NPCs

## Returns

- `long`: The count of NPCs
