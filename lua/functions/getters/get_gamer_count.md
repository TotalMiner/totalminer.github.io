
# (Official) LUA Scripting Documentation

## get_gamer_count

Get the total number of valid gamers in the session.

___

Spec:

```lua
get_gamer_count()
```

## Returns

- `long`: The number of gamers

___

## Example

```lua
local total_gamer_count = get_gamer_count()
```
