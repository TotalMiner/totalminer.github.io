
# (Official) LUA Scripting Documentation

## get_permission

Query if the context player has a particular permission.

___

Spec:

```lua
get_permission(
	string name)
```

## Parameters

- `name`: The permission being queried

## Returns

- `bool`: True if the player has the permission

___

Name can be one of the following:
- Adventure
- Edit
- Creative
- Fly
- Map
- Save
- Admin
- Grief
- VoiceChat
- Spectate
- SystemShops
- ViewScripts
- TextChat
