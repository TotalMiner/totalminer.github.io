
# (Official) LUA Scripting Documentation

## get_skill_xp

Get the current xp of a specified skill of the context actor.

___

Spec:

```lua
get_skill_xp(
	string skill)
```

## Parameters

- `skill`: 

## Returns

- `double`: The skill xp (experience points)

___

## Example

```lua
local xp = get_skill_xp("mining")
```
