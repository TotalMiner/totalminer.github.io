
# (Official) LUA Scripting Documentation

## get_actor_avatar

Get the avatar of the context actor.

___

Spec:

```lua
get_actor_avatar()
```

## Returns

- `string`: Returns the actor's avatar.

___

## Example

```lua
notify("Your current avatar is: "..get_actor_avatar())
```
