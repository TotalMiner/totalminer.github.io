
# (Official) LUA Scripting Documentation

## get_item_name

Get the name of an item.

___

Spec:

```lua
get_item_name()
```

## Returns

- `string`: The item name.

___

## Example

```lua
notify("You collected a "...get_item_name(item_id)..."!")
```
