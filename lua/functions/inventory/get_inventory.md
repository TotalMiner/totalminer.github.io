
# (Official) LUA Scripting Documentation

## get_inventory

Get the total count of an item in the context actors inventory.

___

Spec:

```lua
get_inventory(
	long item_id)
```

## Parameters

- `item_id`: A valid item_id is required

## Returns

- `long`: The total number of item_id's in inventory
