
# (Official) LUA Scripting Documentation

## get_clan_history

Get a value from the context player's clan history.

___

Spec:

```lua
get_clan_history(
	string key)
```

## Parameters

- `key`: The key/name of the history record

## Returns

- `long`: The history value

___

## Example

```lua
local member_count = get_clan_history("member count")
```
