
# (Official) LUA Scripting Documentation

## get_sys_history

Get a value from system history.

___

Spec:

```lua
get_sys_history(
	string key)
```

## Parameters

- `key`: The key/name of the history record

## Returns

- `long`: The history value

___

## Example

```lua
local temp = get_sys_history("temperature")
```
