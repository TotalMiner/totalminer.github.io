
# (Official) LUA Scripting Documentation

## set_sys_history

Set a value in system history.

___

Spec:

```lua
set_sys_history(
	string key,
	long value)
```

## Parameters

- `key`: The key/name of the history record
- `value`: The value to set the history record to. Negative values are supported
